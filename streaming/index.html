The stack

Platform
Twitch primary Restream to YouTube + Kick once it makes sense. Working toward Twitch Affiliate - 50 followers, 500 broadcast min, 7 unique days, 3 avg concurrent viewers (30-day rolling).
Capture
OBS Studio Free, native, low CPU. Single browser source per scene.
Overlays
StreamElements Cloud-rendered overlays + alerts + tip page. JWT auth. Custom CSS over the default tip UI. CPU stays at 3-5% on stream rig.
Bots
SE Chatbot + ZOE StreamElements for Twitch-native commands. ZOE for ZAO-aware replies, scheduling, and on-chain context. !zao, !fid, !bcz, !schedule.
Realtime
SE realtime WebSocket Live tip / follow / sub / cheer / raid feed into the BCZ Farcaster mini app. The mini app shows a live indicator when the stream is running.

Where this plugs into the ZAO ecosystem

Twitch as top-of-funnel for Farcaster. The snap embed at lank.online softens the JFS auth requirement so anonymous web viewers can interact with a snap before signing into Farcaster - useful for moving Twitch viewers onto the protocol with less friction.
